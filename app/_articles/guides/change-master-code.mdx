## Troubleshooting

**New code doesn't work:**
- Wait 2 minutes  - sync time
- Enter slowly  - check beeps
- Verify in app  - shows new code
- Try old code  - may not have changed

**Locked out:**
- Use backup key
- 9V emergency power
- Factory reset  - last resort
