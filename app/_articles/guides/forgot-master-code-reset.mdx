After reset, set new master code immediately.

## Recovery Methods

### Method 1: View Master Code in App (Best Option)

**If you still have app access:**

```
This is the easiest solution - you may have forgotten code but app still knows it.

‚ñ° Open smart lock app
‚ñ° Select your lock
‚ñ° Go to Users / Codes / Settings
‚ñ° Find "Master Code" or "Admin Code"
‚ñ° Tap "Show" or "View Code"

‚ñ° Code displayed: ________

‚ñ° Test code on keypad immediately
‚ñ° Write down code in secure location
```

**Time:** 1-2 minutes

**This works if:**
- ‚úÖ You have admin access to app
- ‚úÖ Lock is online
- ‚úÖ You remember app login

**This doesn't work if:**
- ‚ùå Lost phone  - no app access
- ‚ùå Forgot app password
- ‚ùå Lock offline  - can't retrieve from cloud

### Method 2: Use Backup Physical Key

**If lock has key backup:**

```
Most smart locks include physical key as backup.

‚ñ° Locate backup key
  - Should be on your keyring
  - OR: Given to you at installation
  - OR: Stored in safe location

‚ñ° Find keyway on lock
  - Usually below handle (Schlage, Kwikset)
  - Above handle (Yale)
  - Behind decorative cap (some models)

‚ñ° Insert key and unlock
  - Works even with dead battery
  - Mechanical override

‚ñ° Once inside:
  - Access app to view/reset master code
  - OR: Perform factory reset from interior
```

**Time:** 2-5 minutes

**If you lost backup key:**
- Order replacement from manufacturer  - $10-20
- Call locksmith to pick lock  - $100-150
- Factory reset from interior  - if accessible

### Method 4: Factory Reset (Last Resort)

**Warning:** Erases ALL codes and settings

**Only do this if:**
- ‚ùå No app access
- ‚ùå No backup key
- ‚ùå Other methods failed

**General factory reset process:**

```
From INTERIOR side:

‚ñ° Remove battery cover
‚ñ° Locate reset button

  - May be labeled "Reset" or "Program"

‚ñ° Press and HOLD reset button

  - Release when beeps/flashes

‚ñ° Wait 30 seconds (lock reinitializes)

  - All codes erased
  - All users removed
  - All settings reset
  - Unpaired from hub/app

‚ñ° Set new master code immediately
  - Follow manufacturer instructions
  - Usually: Enter programming mode ‚Üí Set code

‚ñ° Re-pair with hub/app

‚ñ° Add back users/codes
```

**Time:** 10-20 minutes

## Brand-Specific Reset Instructions

### Yale Assure Lock

**Factory reset:**
```
1. Remove battery cover (interior)
2. Press and hold program button (gear icon)
3. While holding, remove one battery
4. Wait 10 seconds
5. Reinsert battery (still holding button)
6. LED flashes green = Reset complete
7. Release button

All codes erased. Set new master code in app.
```

### Schlage Encode

**Factory reset:**
```
Option A (from interior):
1. Remove battery cover
2. Press and hold program button (behind cover)
3. Hold for 20 seconds until beeps
4. Release
5. Lock resets

Option B (from exterior):
1. Enter programming code (if you know it)
2. Press Schlage button + 0 simultaneously
3. Hold until beeps
4. Lock resets
```

### Kwikset SmartCode

**Factory reset:**
```
1. Remove battery cover
2. Press and hold program button
3. While holding, reinsert batteries
4. Keep holding for 30 seconds
5. LED flashes red = Reset complete
6. Release

Lock is now factory default. Set new master code:
- Press program button once
- Enter new master code
- Press lock button
- Re-enter master code
- Press lock button  - beeps = Confirmed
```

### August Smart Lock

**Factory reset:**
```
1. Remove lock from door (or access interior)
2. Hold down reset button (small hole, use paperclip)
3. Hold for 10-15 seconds
4. LED turns red, then cycles colors
5. Release when LED stops cycling

Lock unpaired. Re-pair with app:
- App will create new codes
- No manual master code setting needed
```

### Z-Wave Locks (General)

**Factory reset (for most Z-Wave locks):**
```
Option A: Through hub
1. Hub ‚Üí Devices ‚Üí Lock ‚Üí Settings
2. "Remove Device" or "Factory Reset"
3. Follow prompts

Option B: Z-Wave Exclude
1. Hub ‚Üí Add Device ‚Üí Z-Wave ‚Üí Exclude
2. Press lock button (varies by model)
3. Lock excluded (reset)
4. Re-pair as new device
```

## After Factory Reset

### Immediate Steps

```
‚ñ° Set new master code right away
  - Don't delay (lock may be unsecured)
  - Choose 6-8 digit code (if supported)
  - Write down in password manager

‚ñ° Test new master code 3 times
  - Ensure it works before leaving

‚ñ° Re-pair with hub/app
  - Follow pairing instructions for your lock
  - Test app access

‚ñ° Add back family codes
  - Recreate user codes
  - Test each code

‚ñ° Reconfigure settings
  - Auto-lock timeout
  - LED/audio settings
  - Notifications

### Prevent Future Lockouts

**Document codes securely:**
```

‚òë Give backup key to trusted person

  - Don't rely on single account

  - Store in secure cloud (not phone gallery)

  - Last resort only

## Common Questions

**Q: Will factory reset work from outside?**

A: Usually NO. Most locks require interior access for factory reset (security feature). Exception: Some allow reset via app if you have access.

**Q: Can I reset without erasing codes?**

A: No. Factory reset erases everything. That's the security - can't reset without deleting data.

**Q: Will I need to re-pair with hub?**

A: Yes. Factory reset unpairs lock from all networks. You'll need to pair again as if new lock.

**Q: How long does reset take?**

A: Reset itself: 30 seconds. Total time including re-setup: 10-20 minutes.

**Q: Can manufacturer help me recover code?**

A: Usually NO for security reasons. They can't access your codes. Exception: Some cloud-based systems may allow account recovery.
